Pattern Drafting Terminology

Block

A block is a template used to pattern draft different patterns.

Pattern

A pattern is used as a template to cut ouf fabric to sew.

What is Pattern Drafting?

Pattern drafting is the process of taking a person's measurements and creates blocks which fit them. It is a very important part of fashion design!

Blocks vs Patterns

This app creates blocks which are used as templates. You trace the blocks and manipulate them to create patterns. To create a pattern out of you must add seam allowances!
